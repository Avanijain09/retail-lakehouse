Production-grade end-to-end data engineering project using:

- PostgreSQL 15
- Python 3.11
- Great Expectations
- Tableau

Raw Data → Bronze → Silver → Gold → PostgreSQL → Tableau

- Data ingestion pipeline
- PySpark transformations
- Data quality validation

- Forecasting and ML
